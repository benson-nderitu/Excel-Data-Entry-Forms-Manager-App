## Running the App

We are using [uv](https://github.com/astral-sh/uv) as our Python package manager for faster and more reliable dependency management.

### Install dependencies

```sh
uv pip install -r requirements.txt
```

### Run the app

```sh
uv venv
uv pip install -r requirements.txt
python main.py
```
